# Sales Analytics Dashboard (SQL + Excel)

## Project Overview
This project demonstrates an end-to-end sales analytics workflow using SQL
for data analysis and Excel for dashboard visualization.

## Tools Used
- SQL (data extraction, aggregation)
- MS Excel (Pivot Tables, Slicers, Dashboard)

## Key Insights
- Top 20% customers contribute nearly 80% of total revenue
- August shows the highest seasonal sales
- Customers aged 20–29 and 50–59 generate the most revenue
- Laptops and T-shirts are top-performing products
